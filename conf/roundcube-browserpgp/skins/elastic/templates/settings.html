Browser PGP

Your private key stays in this browser only (passphrase-encrypted in local storage) and never reaches the server. Encryption and decryption happen in your browser.

Import an existing key

Paste an ASCII-armored PGP key. A private key is stored in this browser; a public key is stored as your public identity.

Publish / export your public key

Publishing stores only your public key on this box so others (and the at-rest encryption feature) can encrypt to you.

Add a contact's public key so you can encrypt to them. Local accounts on this box are looked up automatically.
